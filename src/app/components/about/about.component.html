About the app

The Student project database is a SPA built as a course project during the Angular course @SoftUni in February/March 2024.

The app has educational and demo purposes only. It solely aims to showcase the learnt techniques in using the Angular framework to build interactive web applications.

Back-end technologies are outside of the course scope. The app is intended to use a slightly modified RESTful API provided and built by the SoftUni academy for the workshop exercises.

HTML/CSS are outside of the course scope. Futhermore, they're part of the next course in the SoftUni academy learning path. The SPDB app is built with only basic knowledge and little experience in HTML/CSS. Bear with the basic and non-responsive design.

Enjoy using the app!
